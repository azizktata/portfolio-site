Project Overview

Van Rental GUI is a modern web application designed to simplify the process of renting vans. Built with React, React Router 6, HTML, CSS, and JavaScript, this application provides a simple and intuitive user interface for finding and booking vans.

To simulate backend interactions during development, I used Mirage JS to mock the server, allowing for better front-end testing and functionality without needing a real backend in place.

The app is fully responsive and works on all devices. The app is hosted on Netlify and the code is available on GitHub.

Feel free to check out the project by visiting the live link!
